# Voronoid - Mood & Energy Tracker Website

Website for Voronoid, a mood and energy tracking app that visualizes your daily entries as beautiful Voronoi diagram calendars.

## Features

- Daily mood and energy tracking
- Voronoi calendar visualization
- Custom color palettes
- Insights and pattern analysis
- Export and share functionality
- 100% private, on-device storage by default

## Project Structure

- `index.html` - Main landing page
- `privacy.html` - Privacy policy
- `terms.html` - Terms of service
